Scratch Project 9/7/16 (CSE)

This was the first project I did in my computer science engineering course. In this project we were able to using Scratch; an online programming language that people can use to create interactive stories, games, and animations I have to admit it was not my best idea and didn't turn out too well. But the objective was to create a game or a story . My job in this project was to work on code and draw the pictures for our project. Some of the features you had to have were user interaction, an objective, and that the game should have included different states in order for the user to experience variety. In the process of creating our story I learned how to compromise and put me and my partner's ideas into one. We tried to make our project very unique so we hand-drew the characters in our story, and tried to come up with a "unique" storyline. This project really helped me explore the different features of Scratch, and made me understand the concept of using different blocks of code to create something much larger.
